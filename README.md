# C++ Learning Repository

A comprehensive collection of C++ learning materials and practice programs, covering fundamental concepts to advanced topics including Object-Oriented Programming (OOP), Data Structures, and Problem Solving.

## 📁 Repository Structure

### 1. **C++ Basics** (`/C++` folder)
Core C++ programming concepts and fundamentals:

- **calculator.cpp** - Basic calculator implementation
- **Functions.cpp** - Function definition, declaration, and calling
- **if-else.cpp** - Conditional statements and branching
- **class.cpp** - Introduction to classes and objects
- **oops.cpp** - Object-Oriented Programming concepts
- **swap.cpp** & **swap2.cpp** - Swapping values and techniques
- **ptr_to_derivedcls.cpp** - Pointers to derived classes (inheritance)
- **main.cpp** - Main entry point program

### 2. **Object-Oriented Programming** (`/OOPs` folder)
Advanced OOP concepts and practical implementations:

- **main.cpp** - OOP examples and main programs
- **DSA.cpp** - Data Structures and Algorithms implementations

#### **Dynamic Allocation** (`/OOPs/Dynamic allocation/`)
Memory management with dynamic allocation:
- **arr.cpp** - Dynamic array allocation and manipulation
- **dyn_constructor.cpp** - Constructors with dynamic memory allocation

#### **File Handling** (`/OOPs/File handling/`)
File I/O operations in C++:
- **main.cpp** - File reading and writing operations
- **file.txt** - Sample data file for practice

#### **Pointer Concepts** (`/OOPs/folder/`)
Pointer operations and advanced pointer usage:
- **ptr.cpp** - Basic pointer operations
- **pointer_to_constant.cpp** - Pointers to constant values
- **ptr_to_class.cpp** - Pointers to class objects
- **Complement.cpp** - Bitwise complement operations

#### **Operator Overloading** (`/OOPs/Overloading/`)
Implementing custom operator behaviors:
- **oper_overload.cpp** - Operator overloading examples
- **function.cpp** - Function overloading

### 3. **Problem Solving** (`/Problem Solving/` folder)
Practical coding problems and solutions:
- **countdigit.cpp** - Count digits in numbers
- **integers.cpp** - Integer-based problems
- **largest.cpp** - Finding largest element/number

## 🚀 Getting Started

### Prerequisites
- C++ compiler (g++, MSVC, or Clang)
- Code editor (VS Code, Visual Studio, etc.)
- Basic command line knowledge

### Compiling and Running

**Using g++:**
```bash
g++ -o program_name filename.cpp
.\program_name.exe
```

**Or use the VS Code build task:**
- Press `Ctrl+Shift+B` to build the active file

## 📝 Notes

- All programs are designed for educational purposes
- Each file focuses on specific C++ concepts
- Progress from basic to advanced topics in the folders' order
- Experiment and modify the code to deepen understanding

## 💡 Recommended Learning Path

1. Start with basic C++ concepts in the `/C++` folder
2. Move to `/OOPs/folder/` for pointer fundamentals
3. Explore `/OOPs/Dynamic allocation/` for memory management
4. Study `/OOPs/Overloading/` for advanced concepts
5. Practice with `/Problem Solving/` problems
6. Implement file operations using `/OOPs/File handling/`

## 🔧 Building & Execution

The repository includes VS Code build configuration:
- **Build Command**: `g++.exe -fdiagnostics-color=always -g ${file} -o ${fileDirname}\${fileBasenameNoExtension}.exe`
- **Shortcut**: `Ctrl+Shift+B`
